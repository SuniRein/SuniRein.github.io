七日创站记 第五日 域名与备案

发表于 2024-11-22 分类于 七日创站记 本文字数： 2.2k 阅读时长 ≈ 9 分钟

阅前须知

本文主要讲述如何获取个人域名、配置域名解析以及进行ICP备案和公安备案。 你的个人博客需要部署在国内的服务器上，否则将无法完成备案操作。

域名购买

在拥有一个自己的域名后，你便可以将域名与自己的博客绑定到一起，通过域名来访问个人博客了。

首先，我们要先到域名注册商处购买域名。 这里我推荐直接在你的云服务器厂商处购买域名，后续的操作会方便一些。

点击这里可以跳转阿里云购买域名：传送门。

根据工信部2017年发布的 《工业和信息化部关于规范互联网信息服务使用域名的通知》 规定： 自2018年1月1日起，在国外域名注册商注册的域名，必须转入国内，才可以正常备案。

有些地方的备案机构还对可以备案的域名后缀有限制，在购买域名前， 请先了解你户口所在地或居住地的域名备案相关规定。

选择好合适的域名后，填写完信息模板，等待审核结束，就能完成域名的购买了。 请确保在购买域名时填写信息的真实性，否则将对后续备案产生影响。

注意，域名的注册费用和续费费用通常不一致。 一般来说，续费的费用会高一些。 如果后续没有更换域名的需求，最好一次性购买足够的年份，减少经济成本。

ICP备案

由于域名实名认证约3天，才会将实名认证信息入库管局，因此建议在实名认证完成3天后，再申请ICP备案，否则阿里云初审时间将会延长。

根据我国现行法律规定，当网站托管在中国内地服务器上时，必须按照备案主体所在省市的管局规则先进行ICP备案，才能对外提供服务。

备案前准备

在备案前，请先检查下你是否满足备案的前提条件：

  • 检查域名是否满足要求，具体可以参考 域名准备与检查。

  • 检查云服务器是否满足要求： 中国内地的所有地域、计费方式为包年包月（包含自动续费3个月及以上）、已分配公网IP地址。

在准备好后，点击此处跳转阿里云备案入口：传送门，选择自助备案后开始备案。

备案信息填写

阿里云上备案信息的填写要求已经非常详尽了，只要按照要求填写即可。

这里我主要补充一些要注意的细节：

  1. 备案地区的选择应为户口所在地或者居住证上的地址（学生证等不能作为居住证的替代）。

  2. 网站的名称要能够体现出网站的内容，备注信息要具体详尽一点，不能笼统含糊。 由于博客类的网站备案需要特殊的前置审批，建议网站说明的时候不要出现博客等字眼。

  3. 主办者信息中地区一栏，要与购买域名时创建的个人信息模版相一致。

备案审核

提交备案信息后，只需要等待审核通过即可，大概流程如下：

  1. 阿里云会在1个工作日完成初审，这段时间内请注意接听来自阿里云的电话。

  2. 阿里云初审通过后约5分钟会收到工信部的核验短信，需要在收到短信的24小时内， 前往ICP/IP地址/域名信息备案管理系统， 完成短信核验。

  3. 备案信息提交管局审核，最长20个工作日就能得到审核结果。 如果备案成功，你会受到一条包含ICP备案号的短信通知。

悬挂ICP备案号

ICP备案成功后，需要在网站首页底部悬挂ICP备案号并生成链接指向工信部网站。

编辑_config.next.yml，找到footer.beian项，填写以下信息： 

footer:
  beian:
    enable: true
    icp: 粤ICP备2024324598号 # 这里填写你的备案号

域名解析

完成ICP备案后，便可以将域名绑定到你的博客上了。

阿里云服务器解析

进入阿里云云解析DNS控制台， 选择要解析的域名，添加一条A类型解析记录， 记录值填写云服务器的公网IP地址，其他选项保持默认即可。

等待几分钟后即可用域名来访问你的个人博客。

Github Page解析

在DNS控制台添加一条CNAME类型记录，解析请求来源选择境外，记录值为你Github Page的地址。

在博客主目录source文件夹下创建一个名为CNAME的文本文件，其内容为你个人域名的地址，然后重新deploy。

这样就完成了Github Page与你个人域名的绑定。 当国内IP访问你的网站时，会解析到阿里云的云服务器； 当国外IP访问你的网站时，则会解析到Github Page。

可以用阿里云的网络拨测工具 来验证域名解析的有效性。

请不要在Github Page的管理界面直接添加自定义域名。 这种方式添加的域名解析会在你下次部署博客时被清除。

SSL

要通过HTTPS协议来访问你的博客，还需要部署SSL证书。

签发证书

点击这里进入阿里云数字证书管理服务， 选择SSL证书管理 > 个人测试证书 > 创建证书，完成SSL证书的签发。

阿里云为个人提供了20张免费的SSL证书，每张证书的有效期为3个月， 总的有效期有5年，对于个人博客来说已经够用了，只需要注意每3个月得重新签发一次证书。 如果你有足够的经济能力，也可以选择付费的SSL证书。 不过除了有效期会长一点外，对于个人博客来说免费证书和付费证书没有区别。

部署证书

证书签发好后，需要部署证书到Nginx服务器上才能使用。

点击证书管理 > SSL证书管理 > 下载，选择服务器类型为Nginx的那一项，单击下载。

解压下载得到的压缩包，会得到证书文件<address>.pem和证书私钥文件<address>.key这两个文件。 我们需要将它们部署到云服务器上。

# 使用ssh将文件传输到服务器，并放到/etc/nginx/cert文件夹下
# 你也可以根据自己服务器的配置，放到任意想要的路径
scp <addredd>.pem <address>.key aliyun.root:/etc/nginx/cert

# 使用ssh连接云服务器
ssh aliyun

# 编辑网站配置文件
sudo vim /etc/nginx/site-available/<name>.conf

# 编辑完后需重启nginx服务
sudo systemctl restart nginx

修改的配置文件内容如下，主要是添加了443端口的解析。 

server {
    listen 80 http2;
    listen [::]:80 http2;
    server_name blog.sunirein.tech; # 这里填写域名地址

    root /var/repo/hexo; # 你的hexo仓库地址
    index index.html index.htm index.nginx-centos.html;
    error_page 404 /404/;

    location / {
        try_files $uri $uri/ =404;
    }

    location = / {
        try_files /index.html =404;
    }
}

server {
    listen 443 ssl http2;
    listen [::]:443 ssl http2;
    server_name blog.sunirein.tech; # 这里填写域名地址

    ssl_certificate /etc/nginx/cert/blog.sunirein.tech.pem; # 证书文件路径
    ssl_certificate_key /etc/nginx/cert/blog.sunirein.tech.key; # 证书撕咬文件路径

    root /var/repo/hexo; # 你的hexo仓库地址
    index index.html index.htm index.nginx-centos.html;
    error_page 404 /404/;

    location / {
        try_files $uri $uri/ =404;
    }

    location = / {
        try_files /index.html =404;
    }
}

开放443端口

HTTPS协议需要用到443端口，还需要在阿里云控制台 的云服务器安全组规则中开放443端口。

以上步骤完成后就可以直接用HTTPS协议来访问你的博客了，可以在浏览器上验证下是否部署成功。

公安备案

完成ICP备案后，还需要在开通网站（即网站对外提供访问服务）之日起的30日内提交公安联网备案申请。

  1. 进入全国互联网安全管理服务平台，完成登录注册。

  2. 点击业务办理 > 开办主体管理 > 新增主体，完成备案主体的创建。 注意，与ICP备案不同，这里的地址需要你如实填写你的居住地址。 有些地方的公安备案还可能需要你线下到公安局登记，请勿填写虚假地址。

  3. 点击业务办理 > 网站业务 > 新办网站申请，按照要求完成信息填写。 "是否提供互联网交互服务要"选"是"，"网站类型"要选"用户内容咨询类（E4）"。

在公安备案审核期间，需要保证你的博客能够被正常访问到。

相比ICP备案，公安备案的流程会简单很多，一般在提交申请后的两三天内就能通过备案了。

在备案审核通过后，记得将备案信息放到个人博客上。

编辑_config.next.yml，填写完整的备案信息： 

footer:
  beian:
    enable: true
    icp: 粤ICP备2024324598号 # ICP备案号
    gongan_id: 33010602013756 # 公安备案ID（即数字部分）
    gongan_num: 浙公网安备33010602013756号 # 完整备案号
    gongan_icon_url: /images/gongan.webp # 公安备案图标
